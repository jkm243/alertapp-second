# Polices Inter

Pour utiliser la police Inter dans l'application, téléchargez les fichiers de police depuis [Google Fonts](https://fonts.google.com/specimen/Inter) et placez-les dans ce dossier.

## Fichiers requis

- `Inter-Regular.ttf` - Police normale
- `Inter-Medium.ttf` - Police medium (poids 500)
- `Inter-SemiBold.ttf` - Police semi-bold (poids 600)
- `Inter-Bold.ttf` - Police bold (poids 700)

## Instructions

1. Visitez [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)
2. Cliquez sur "Download family"
3. Extrayez les fichiers TTF
4. Placez les fichiers dans le dossier `assets/fonts/`
5. Exécutez `flutter pub get` pour mettre à jour les dépendances

## Alternative

Si vous préférez utiliser une police système similaire, vous pouvez modifier le `pubspec.yaml` pour utiliser une police par défaut du système.
